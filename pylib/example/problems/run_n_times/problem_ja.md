# run N times サンプル問題

## 問題文

長さ $N$ の順列全体の集合 $S_N$ に対して全ての元に対し $N$ 回の適用で元に戻る全単射写像 $f: S_N \rightarrow S_N$ を構成してください。

## ジャッジの方法について

この問題は以下のようにジャッジされます。

1. ジャッジ側が正整数 $N$ と、長さ$N$ の順列 $P_0$ を定める。
2. 提出コードを $N$ 回実行する。
    - $i \,(1 \le i \le N)$ 回目の実行では、正整数 $N$ と、順列 $P_{i-1}$ が標準入力で与えられる。
    - あなたは、構成した写像を $P_{i-1}$ に適用して得られる順列 $P_{i}$ を標準出力から出力する。
    - ここで出力された順列が、$i + 1$ 回目の実行で入力される順列となる。
    - 各回の実行では、それより前に入力された順列や、何回目の実行かを知ることはできない。
3. $N$ 回の実行が終わった後、出力された順列に対してジャッジをする。
    - $P_0 = P_N$ かつ、 $P_1, \cdots, P_N$ が相異なる長さ $N$ の順列となっている場合、またその場合のみ AC と判定される

## 制約

- $N \le 2 \times 10^5$
- $P_0$ は $(1, 2, \cdots, N)$ の順列
- 入力は全て整数

## 入力

各実行での入力は以下のように与えられる。

> $N$
> $p_1 \,\cdots \, p_N$

ただし、 $p_i (1 \le i \le N)$ は入力で与えられる順列 $p$ の $i (1 \le i \le N)$ 番目の要素である。

## 出力

各実行では次の形式で順列を出力せよ。

> $p_1 \,\cdots \, p_N$

ただし、 $p_i (1 \le i \le N)$ は出力する順列 $p$ の $i (1 \le i \le N)$ 番目の要素である。

## サンプル

初めに、ジャッジが正整数 $N = 2$ と、順列 $P_0 = (1, 2)$ を定める。

### 1回目の実行

| 入力 | 出力| |
| ---- | ---- | ---- |
| `2` |  | $N = 2$ |
| `1 2` |  | $P_0 = (1, 2)$|
|  | `2 1` | あなたは $P_1 = f(P_0) = (2, 1)$ と定めた。|

### 2回目の実行

| 入力 | 出力| |
| ---- | ---- | ---- |
| `2` |  | $N = 2$ |
| `2 1` |  | $P_1 = (2, 1)$|
|  | `1 2` | あなたは $P_2 = f(P_1) = (1, 2)$ と定めた。|

### ジャッジ

出力された順列 $P_1, P_2$ はどちらも長さ $2$ の順列であり、$P_0 = P_2$ かつ、$P_1 \ne P_2$ を満たすため AC と判定される。

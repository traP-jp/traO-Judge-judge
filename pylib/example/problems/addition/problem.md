# Addition(sample problem)
You are given two integers L and R. Output the sum of L and R.
## Input
- The input is given from Standard Input in the following format:
```
L R
```
## Output
- Print the sum of L and R.
## Constraints
- $1 \leq L \leq R \leq 100$
- $L$ and $R$ are integers.
---

### Sample Input 1
```
1 2
```
### Sample Output 1
```
3
```
The sum of 1 and 2 is 3.

---

### Sample Input 2
```
72 108
```
### Sample Output 2
```
180
```
The sum of 72 and 108 is 180.
